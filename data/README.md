# Data Layout

本目录用于承载仓库内可公开的最小数据材料与数据说明。

## 目录约定

- `sample/`: 可提交到 GitHub 的小样本，仅用于字段格式演示。
- `raw/`: 分源抓取或检索结果，默认忽略，不纳入版本控制。
- `intermediate/`: 清洗、去重、合并后的中间结果，默认忽略。
- `cache/`: 缓存文件，默认忽略。
- `private/`: 任何不适合公开提交的数据或临时搬移数据，默认忽略。

## 说明

- 当前仓库不包含全量原始数据和全量清洗结果。
- `data/sample/sample_papers.jsonl` 为结构演示样例，不代表真实完整数据集规模。
